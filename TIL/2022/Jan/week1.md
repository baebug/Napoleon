### - 알고리즘

- 기본 수학 단계로 들어왔는데 진짜 수학적인 사고력을 요하는 문제들이 나오기 시작했다.
- 등차수열처럼 단계적으로 수가 증가하고 어떤 숫자의 구간을 찾는 문제?들

```python
# ex. 1/1, 1/2, 2/2, 1/3, 2/3, 3/3, 1/4, ... 일 때, n번 째 수는 무엇인가?
idx = 0

while n > 0:
	idx += 1
	n -= idx
	
print(idx)

# n = 3 이면 idx = 2
```

- 이런 식으로 `while` 을 사용해서 구현했는데, 괜찮은 방법인 것 같다.
